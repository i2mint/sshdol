sshdol.tests.utils_for_testing

Utils for testing.

sshdol.tests.utils_for_testing.empty_test_store(store, *, store_as_expected=<function _is_the_test_folder>, keys_as_expected=<function _keys_as_expected>)[source]

A function that deletes all items of a store, after verifying a condition on the keys. The raison d’être of this function is to not mistakingly empty the wrong store.

sshdol.tests.utils_for_testing.sort_keys_deepest_first(keys)[source]

Return keys sorted by deepest path first, then reverse-alphabetically.

This ensures files in deeper directories are deleted before their parent directories, avoiding “directory not empty” errors on stricter filesystems/servers.
